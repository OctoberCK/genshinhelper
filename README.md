# genshinhelper

## 注意事项
1. 本Javascipt脚本适用于iOS的 `Scriptable` APP，使用前请下载。   
2. 遇到问题发issue前请检查操作的步骤，特别是注意 `uid` 的填写是否正确。
3. 使用前请先在js脚本中添加配置(复制后填写对应的 `uid` 和 `cookie` )，必须加入到脚本标示的位置：  
```
config[0] = "uid"
config[1] = "cn_gf01"
config[2] = "cookie"
```
4. 支持全服务器  
B服： `config[1] = "cn_qd01"`   
亚服： `config[1] = "os_asia"`  
欧服： `config[1] = "os_euro"`  
美服： `config[1] = "os_usa"`  
港澳台服： `config[1] = "os_cht"`  

5. 或者使用iOS快捷指令一键安装对应的配置(需要iOS15.0及以上版本)。  
**请务必检查并开启，否则无法使用**
